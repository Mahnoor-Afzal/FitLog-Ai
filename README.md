# FitLog-Ai
AI-Fitness-Workout-Planner

A comprehensive Flutter fitness tracker app featuring AI coaching, real-time workout tracking, cloud persistence, and detailed progress reporting.

## Features
- **AI Coaching**: Get personalized workout plans and diet suggestions.
- **Workout Tracking**: Real-time tracking of exercises with timer.
- **Progress Reports**: Detailed charts for calories burned and weight history.
- **Cloud Sync**: Data persistence using Firebase Firestore and Storage.
- **Custom Workouts**: Create and save your own workout routines.
